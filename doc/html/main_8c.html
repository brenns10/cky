Read a file completely into memory.

This function allocates space for a file's contents and reads it into memory. The function also requires a parameter in which to store the number of chars it has allocated in the buffer, so the memory can be accounted for when freed. This function is independent of encoding.

A function that tests the basic construction of a simple FSM.

Using the actual creation functions, creates a FSM to accept a string with an even number of a's and b's. Currently, the function simulates it nondeterministically, even though the machine is deterministic.
